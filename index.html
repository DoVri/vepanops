Windows Setup

# 1. Open Notepad as Administrator
File Path: C:\Windows\System32\drivers\etc\hosts
# 2. Add these lines at the very bottom:
SERVER IP growtopia1.com
SERVER IP growtopia2.com
SERVER IP www.growtopia1.com
SERVER IP www.growtopia2.com
# 3. Save (Ctrl+S) and open Growtopia.
